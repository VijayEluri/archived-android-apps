
This application has three main views:

- Configuration:

Allows the user to select which Cineworld cinema they are interested in.

Allows the user to choose which action to take when the widget is clicked:
	- Open booking screen
	- Show list

- Widget:

Cycles through each film that is currently playing in the selected Cineworld
cinema showing the next show time.


- Main:

Allows the user to see all showings for each film by default in the 
currently selected cinema.

Clicking on a showing opens the Cineworld booking screen.

Clicking on a film title opens the Cineworld film info.
