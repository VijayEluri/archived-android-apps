Initiative Tracker

To get started use the menu to add toons.

Tap a character to edit its initiative and add/remove effects. Changes to initiative will reorder the toons. The effect icons can mean whatever you want them to mean.

Use the "Next Round" option in the Combat menu to progress the round, all current effects counters will be incremented by 1.

Use the "New" option in the Combat to clear all effects, ready for a new combat.
